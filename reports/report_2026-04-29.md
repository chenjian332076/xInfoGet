## AI 前沿动态

### 1. OpenAI 模型、Codex 与 Managed Agents 登陆 AWS / Amazon Bedrock

> 来源: OpenAI Official | 2026-04-28

OpenAI 与 AWS 宣布扩大战略合作，面向企业客户在既有 AWS 环境内统一交付三类能力（均为 **limited preview**）：**(1)** 在 Amazon Bedrock 上使用包括 **GPT‑5.5** 在内的 OpenAI 前沿模型；**(2)** **Codex on AWS**：用户可将 Codex 配置为通过 Bedrock 调用模型，兼容既有 AWS 承诺用量与账单体系，首发支持 Codex CLI、桌面应用与 VS Code 扩展；**(3)** **Amazon Bedrock Managed Agents, powered by OpenAI**，面向生产级多步工具调用与工作流，强调与 AWS 安全、合规控制的内置集成。官方指出每周已有 **400 万+** Codex 用户，此举意在缩短从试验到生产的闭环。

**关键观点**: 前沿模型竞争正与「客户已在用的云账户、身份与治理」绑定——**谁在 Bedrock/Azure/GCP 上提供同质 harness，谁就吃掉企业采购默认值**。

### 2. 微软–OpenAI 伙伴关系进入新阶段：多云、非独占许可与分成封顶

> 来源: OpenAI Official | 2026-04-27

双方公布修订后的长期协议要点：**微软仍是 OpenAI 首要云伙伴**，OpenAI 产品默认优先上线 Azure，但若微软无法或选择不支持所需能力则例外；**OpenAI 现可向任意云服务商客户提供全部产品**。微软对 OpenAI IP 的许可延续至 **2032 年**，但改为 **非独占**；**微软停止向 OpenAI 支付收入分成**；**OpenAI 向微软的收入分成延续至 2030 年**，比例不变但 **设有总额上限**，且与 OpenAI 技术是否达到某里程碑脱钩；微软继续作为主要股东参与 OpenAI 成长。声明同时强调双方在数据中心规模扩展、下一代芯片协同、网络安全等领域的持续合作。

**关键观点**: 条款变动标志着 **「单一云独占前沿模型」让位于「多云分发 + 清晰财务封顶」**，与数日内在 AWS 上首发 Bedrock 集成形成互证。

### 3. AC/DC：通过任务–能力协同进化发现 LLM「专家种群」

> 来源: Andrew Dai 等 (Sakana AI), arXiv | 2026-04-16

论文提出 **Assessment Coevolving with Diverse Capabilities (AC/DC)**：在同一运行中协同进化 **LLM 种群**（进化模型合并 crossover、权重奇异值扰动 mutation）与 **合成任务档案**（由「科学家」大模型迭代生成并带评测代码的任务）。核心度量 **Coverage** 刻画种群集体覆盖题型集合的能力，而非单一模型准确率。报告显示种群在多项基准上相对手工专家混合、对照重采样、单体大模型等基线取得 Coverage 提升，并在 Best-of-N 场景下展现互补专长；项目页面与代码见作者主页。

**关键观点**: 若持续扩展「模型多样性 × 任务多样性」比单纯放大单模型更划算，则 **开源与并购合并生态可能涌现「小而专」模型组合**，挑战单一巨型 API。

🔗 [阅读原文](https://arxiv.org/abs/2604.14969)  
🔗 [项目页](https://acdc-llm.github.io/)

### 1. ChatGPT Workspace Agents：面向团队的 Codex 云端共享代理

> 来源: OpenAI Official | 2026-04-22

**Workspace agents** 定位为 GPTs 的演进：由 **Codex** 在云端驱动，可在组织权限内处理报告撰写、代码任务、消息响应等长程工作；支持在 **ChatGPT 与 Slack** 中共建共享、持续迭代同一代理。典型场景包括销售线索研判、Slack 中的内部答疑、月末结算辅助等。治理方面强调工具与动作的分级授权、敏感步骤审批、**Compliance API** 对代理配置与运行的审计可见性；面向 Business / Enterprise / Edu / Teachers **研究预览**，**2026-05-06 前免费**，之后按额度计费。

**关键观点**: 产品叙事从「个人增效」转向 **「可审计的组织级代理资产」**，与同日发布的临床向改进等一同补齐垂直合规叙事。

## 技术趋势与洞察

### 1. Anthropic Project Deal：当代理替你砍价时，「更强模型」溢价可能被感知掩盖

> 来源: Anthropic (Kevin K. Troy 等) |  posted 2026-04-24

Anthropic 在旧金山办公室运行一周 **Slack 内分类广告市集**：69 名员工各持 **100 美元**预算，由 **Claude** 经访谈生成个性化代理，**全程无人工实时干预**完成发帖、议价与成交；真实跑次中 **186 笔**交易、成交额 **逾 4000 美元**。团队在平行实验中对比 **Claude Opus 4.5** 与 **Haiku 4.5**：更强卖方对同类商品可比成交价更高（文中举例同一折叠单车 Haiku 成交 **38 美元**、Opus **65 美元**），买方侧亦有系统性价差；但受访者对公平性的主观评分几乎相同（约 **4.05 vs 4.06**/7），提示 **客观议价劣势未必被用户觉察**。文末讨论代理商业、注意力经济与监管真空。

**关键观点**: 多代理商务不再是思想实验——**模型层级差异会转化为可度量货币差距**，治理焦点应从「能否成交」转向 **透明度与代理代表权披露**。

🔗 [阅读原文](https://www.anthropic.com/features/project-deal)

> 来源: Firstpost（转述 Sam Altman于 X 的发言）| 2026-04-27

据 Firstpost，Sam Altman 在 X 上提出：当前正处于应 **认真反思操作系统、用户界面乃至互联网底层设计** 的时点；未来需要 **对人机代理同等可用** 的协议栈，以便代理独立执行任务、浏览应用而不被「为人类点击设计」的交互所掣肘。报道将该论述与业界推动 **agent-native** 基础设施、意图驱动界面的讨论并列。同一时期另有媒体引用其关于 AGI 与社会影响的短文帖（内容偏宣言性质，此处仅作脉络参照）。

**关键观点**: 领袖层叙事正从「更强模型」延伸到 **重写互操作性标准**——与 MCP、A2UI、云端 harness 的工程浪潮互为表里。
